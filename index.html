CARTA

CARTA is the Cube Analysis and Rendering Tool for Astronomy, a new image visualization and analysis tool designed for the ALMA, the JVLA, and radio telescopes of the future such as the SKA. As the image size from modern telescopes has increased drastically in recent years, contemporary image viewing software face performance issues which degrade the user experience significantly. The mission of CARTA is to resolve this and provide usability and scalability for the future by utilizing modern web technologies and computing parallelization.

Download and Installation

The CARTA Desktop 'local' version is intended for use on a laptop or desktop computer directly connected to a monitor.

  • Local version 1.0-rel download links:
    • macOS 10.12, 10.13, 10.14
    • Red Hat 7 / CentOS7
    • Ubuntu 16.04 LTS, 18.04 LTS

We also provide a 'remote' version for users who wish to run CARTA via the ssh protocol from a remote RedHat6/CentOS6 or RedHat7/CentOS7 headless server.

  • Remote version 1.0-rel download link:
    • Redhat 6/7, CentOS 6/7

    • Mac Installation Instructions: Open the dmg file and drag the CARTA-v1.0.app to the /Applications folder. To run CARTA-v1.0, click on it in the Launchpad, or double click on it in the Finder.

    Linux Installation Instructions (local version): The RedHat7 and Ubuntu versions are provided as AppImages. After downloading, make the AppImage file executable, and simply run it.
    e.g., chmod 755 CARTA-v1.0-ubuntu.AppImage ./CARTA-v1.0-ubuntu.AppImage

    Linux Installation Instructions (remote version): Extract the archive and run the carta executable in your terminal window. This will provide a unique URL. Copy and paste this URL in to your local web browser. When finished, close your browser window, and press 'q' in the terminal window in order to close CARTA.
